# TeamToo

If you want to check Wiki, please click [here](https://github.com/hammii/TeamToo/wiki).
<br>

### What is TeamToo?
```
Team Project Management Android App
```
<br>

### Function
1. Live Chat
2. File Sharing
3. Schedule Management
4. To-do List Management
5. Peer Evaluation
<br>
